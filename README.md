## Available Scripts

In the project directory, you can run:

### `docker-compose up --build`

Builds the app

### `docker-compose run web rails db:create`
### `docker-compose run web rails db:migrate`

Set up the DB

### `docker-compose run web rspec spec/`

Run rspec tests
